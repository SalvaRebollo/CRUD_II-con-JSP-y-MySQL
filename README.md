# CRUD II con JSP y MySQL
Salvador Rebollo Benítez.

## ¿De qué va a ir mi proyecto?
Trabajo para la asignatura programación del grado superior Desarrollo de Aplicaciones Multiplataforma, consiste
en la realizacion de un CRUD funcional sobre una temática a elegir, además de la utilización de dos estructuras de datos 
llamadas ArrayList y HashMap.

Mi CRUD y base de datos serán sobre un videoclub, y usará tres tablas: Socios, películas y alquileres realizados.
